# Exploratory Data Analysis – Netflix

Projeto de Análise Exploratória de Dados (EDA) utilizando um dataset de títulos da Netflix.

## Objetivos
- Analisar a distribuição entre filmes e séries  
- Avaliar a evolução do catálogo ao longo dos anos  
- Identificar países com maior produção de conteúdo  
- Detectar problemas de qualidade nos dados  

## Ferramentas
- Python  
- Pandas  
- Matplotlib  
- Google Colab  

## Principais insights
- O catálogo da Netflix possui mais filmes do que séries.  
- Os Estados Unidos são o país com maior número de títulos.  
- Houve um crescimento acelerado do catálogo a partir de 2010.  
- Existem valores ausentes em colunas como *director* e *country*.  
- A coluna *country* contém múltiplos países por registro, sendo necessário normalizar os dados.  

## Estrutura do projeto
- `EDA_Netflix.ipynb`: notebook com toda a análise exploratória.
